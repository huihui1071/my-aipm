OneBox OS

让一句话先找到路径，再调出真正有用的长期记忆。

OneBox OS 把 AI Task Router 和 Personal Memory OS 放在同一个入口里。你只要输入一句话，系统先判断当前更适合澄清、搜索、写作、分析还是执行，再按需调用长期记忆，让过去的笔记、截图、网页和语音真正参与当前任务。

How To Start
  1. 先直接输入一句话，不用预设模式。
  2. 看系统怎么理解你的任务，并查看引用来源。
  3. 只有当回答缺背景时，再去补长期记忆。
